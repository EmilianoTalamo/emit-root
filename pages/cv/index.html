Mihaly Meszaros

La web consta en un artículo hecho en XHTML estricto y CSS validado por los estándares de la W3C a partir de un tema asignado aleatoriamente. La información tuvo que ser recopilada desde cero y clasificada en diferentes temas para abordarlos en la web y permitir su navegación entre ellos con el uso de anclas.

Una vez recabada la información, se procedió a pensar en el diseño de la web. Debía contar con un header, un cuerpo principal donde se ubicara la información y un footer. Primero se bocetó el esqueleto de la página en un framework hecho a mano. Luego, con herramientas de diseño avanzadas se crearon todos los assets, paletas de colores y la forma visual final de la web.

Recién allí fue cuando se empezó con el código, basándose a partir de los bocetos y frameworks realizados, se creó toda la web utilizando y respetando las reglas semánticas que ofrece HTML. También se utilizó el método de cascada de CSS para lograr un diseño más eficiente.
